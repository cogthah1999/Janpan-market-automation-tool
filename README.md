# 🇯🇵 Japan Market Automation Tool 📈

## 1. Overview
Dự án này là một công cụ tự động hóa giúp thu thập và phân tích dữ liệu thị trường chứng khoán Nhật Bản (Tokyo Stock Exchange) từ Yahoo Finance.

## 2. Key Features
- Auto-Crawling: Tự động lấy giá cổ phiếu theo thời gian thực (Top 10 công ty lớn như Toyota, SoftBank, Sony...).
- Data Analysis: Tính toán biến động phần trăm trong 7 ngày gần nhất.
- Visualization: Tự động vẽ biểu đồ trực quan (Dashboard) so sánh hiệu suất giữa các mã.
- Auto-Export: Lưu trữ dữ liệu vào định dạng Excel để báo cáo.

## 3. Tech Stack
- Language: Python 3.x
- Libraries: yfinance, Pandas, Matplotlib, Seaborn, Schedule.

## 4. How to Use
1. Clone dự án: `git clone https://github.com`
2. Cài đặt thư viện: `pip install -r requirements.txt`
3. Chạy tool: `python src/main.py`
